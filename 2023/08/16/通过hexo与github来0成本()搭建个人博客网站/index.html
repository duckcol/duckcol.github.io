可跟随第一和第二个用以理解，实际操作可参考第三个，第四个是我本人使用的主题，你也可以自行寻找。

你需要的东西

  • 一台window电脑(废话)
  • 魔法
  • 注册Github账号
  • 下载Git
  • 下载nodejs
  • 下载vscode
  • markdown编辑器
  • 查找资料的能力，询问的勇气，一点点耐心

Github账号注册

请记住你的注册的邮箱与账号名

下载Git

https://git-scm.com/download/win

下载nodejs

https://nodejs.org/en/

可按“win+r”打开命令行输入“git -v”与”node -v”检查是否安装成功
